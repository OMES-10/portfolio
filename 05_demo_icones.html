① Inclure Bootstrap Icons dans votre HTML

Ajoutez ce <link> dans le <head> de votre page :

<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">

Ensuite, utilisez simplement la balise <i> avec la classe correspondante :

<i class="bi bi-house-fill"></i>
<i class="bi bi-envelope"></i>
<i class="bi bi-github"></i>

④ Accessibilité (aria-label)

Les icônes seules ne sont pas lisibles par les lecteurs d'écran. Ajoutez aria-label ou un texte masqué :

<!-- Icône décorative (aucune valeur sémantique) -->
<i class="bi bi-star-fill" aria-hidden="true"></i>

<!-- Icône fonctionnelle (bouton / lien) -->
<button aria-label="Envoyer un courriel">
  <i class="bi bi-envelope-fill" aria-hidden="true"></i>
</button>

<!-- Icône avec texte visible (idéal) -->
<a href="mailto:vous@uottawa.ca">
  <i class="bi bi-envelope me-1" aria-hidden="true"></i>
  Contactez-moi
</a>
